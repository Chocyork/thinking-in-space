二、3D Soft-Smoothing 普通版
================================================================================

2.1 模型文件
------------------------------------------------------------------------
文件: /home/qyk/thinking-in-space/lmms_eval/models/llava_onevision_3d_smooth.py
类名: Llava_OneVision_3D_Smooth
注册名: llava_onevision_3d_smooth

方法说明:
- 在 projector 输出后对所有 group 内 tokens 做 soft-smoothing
- 公式: new = (1-α)*original + α*mean(group)
- 保留所有 tokens，只改变特征值
- 2D 空间感知映射 (27x27 -> 新网格)

3.1 模型文件
------------------------------------------------------------------------
文件: /home/qyk/thinking-in-space/lmms_eval/models/llava_onevision_3d_smooth_pruned.py
类名: Llava_OneVision_3D_Smooth_Pruned
注册名: llava_onevision_3d_smooth_pruned
注册位置: /home/qyk/thinking-in-space/lmms_eval/models/__init__.py

方法说明:
- 两阶段剪枝策略:
  Stage1: 视觉编码阶段计算 keep_mask，做 soft-smoothing，但返回完整特征
  Stage2: 在 prepare_inputs_labels_for_multimodal 中物理抹除冗余 tokens
- 只保留每组第一个 representative token
- 序列长度大幅减少 (如 11664 -> 802 tokens)
- EVS Sequential: 重新连续分配 position_ids

五、核心代码逻辑对比
================================================================================

5.1 普通版 (_smooth_features)
------------------------------------------------------------------------
for group in groups:
    indices = get_all_tokens_in_group(group)
    mean_feat = features[indices].mean()
    for idx in indices:
        features[idx] = (1-α)*features[idx] + α*mean_feat
return features  # 所有 tokens 都更新，数量不变

5.2 Pruned 版 (两阶段)
------------------------------------------------------------------------
# Stage1: _smooth_features
for group in groups:
    indices = get_all_tokens_in_group(group)
    rep_idx = indices[0]  # 只取第一个
    mean_feat = features[indices].mean()
    features[rep_idx] = (1-α)*features[rep_idx] + α*mean_feat
    keep_mask[rep_idx] = True
_global_keep_mask = keep_mask
return features  # 返回完整特征，但标记了哪些保留

# Stage2: wrapped_prepare
if _global_keep_mask is not None:
    vis_emb = get_visual_part(embeds)
    vis_emb_pruned = vis_emb[:, _global_keep_mask]  # 物理剪枝
    embeds = concat(text_emb, vis_emb_pruned, text_emb2)
    position_ids = arange(new_seq_len)  # 重新编号
return embeds
